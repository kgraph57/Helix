# 診断支援プロンプトの使用例

このドキュメントでは、診断支援プロンプトの実際の使用例を紹介します。

## 例1: 鑑別診断プロンプトの使用

### シナリオ

救急外来に55歳男性が胸痛を主訴に来院しました。心筋梗塞を含む鑑別診断を検討する必要があります。

### 使用したプロンプト

[鑑別診断プロンプト](../prompts/diagnosis/differential-diagnosis.md)を使用しました。

### 入力内容

```
あなたは経験豊富な臨床医です。以下の患者情報に基づいて、鑑別診断を行ってください。

【患者情報】
- 年齢・性別: 55歳 男性
- 主訴: 胸痛
- 現病歴: 3時間前から持続する胸部圧迫感。左肩への放散痛あり。冷汗、悪心を伴う。安静にしても改善しない。
- 既往歴: 高血圧（10年）、2型糖尿病（5年）、脂質異常症
- 内服薬: アムロジピン5mg、メトホルミン1000mg、アトルバスタチン10mg
- アレルギー: なし
- 家族歴: 父親が50代で心筋梗塞
- 生活歴: 喫煙20本/日×30年、飲酒機会飲酒、デスクワーク
- バイタルサイン: 
  - 血圧: 150/95 mmHg
  - 脈拍: 98 /分（整）
  - 体温: 36.8 ℃
  - 呼吸数: 20 /分
  - SpO2: 96 % (室内気)
- 身体所見: 冷汗あり、顔面蒼白、心音整、雑音なし、呼吸音清、腹部異常なし
- 検査所見: なし（救急外来初診）

【依頼内容】
以下の形式で鑑別診断を提示してください：

1. 最も可能性の高い診断（Top 3）
2. 除外すべき重要な疾患（Must not miss）
3. その他の鑑別診断
4. 推奨される初期対応
```

### AIからの出力例（要約）

AIは以下のような鑑別診断を提示しました。

**最も可能性の高い診断**として、急性冠症候群（ACS）が第一に挙げられました。持続する胸部圧迫感、放散痛、冷汗、リスクファクター（高血圧、糖尿病、喫煙、家族歴）が強く支持する所見です。確定診断には心電図、心筋マーカー（トロポニン）、緊急冠動脈造影が必要とされました。

第二に急性大動脈解離が挙げられました。突然発症の胸痛、高血圧が支持する所見です。造影CTによる確認が推奨されました。

第三に肺塞栓症が考慮されました。呼吸困難の程度は軽いものの、リスクファクターの評価とDダイマー測定、必要に応じて造影CTが推奨されました。

**除外すべき重要な疾患**として、緊張性気胸や心タンポナーデなど、直ちに生命を脅かす疾患が挙げられました。

**推奨される初期対応**として、直ちに心電図実施、静脈ルート確保、酸素投与、心筋マーカー測定、胸部X線撮影が提案されました。循環器専門医への緊急コンサルトの必要性も指摘されました。

### 臨床での活用

この出力を参考に、実際の診療では以下のように対応しました。

直ちに12誘導心電図を実施したところ、II、III、aVFでST上昇を認め、下壁心筋梗塞が強く疑われました。緊急で循環器内科にコンサルトし、緊急冠動脈造影および経皮的冠動脈形成術（PCI）が実施されました。

AIの鑑別診断は、緊急性の高い疾患を適切に優先順位付けし、初期対応の方向性を示すのに役立ちました。

### 学んだポイント

AIプロンプトは、鑑別診断のチェックリストとして有用です。特に多忙な救急外来では、見逃してはいけない疾患を体系的に検討する助けになります。

ただし、AIの出力はあくまで参考情報です。実際の診療では、患者を直接診察し、臨床判断を行った医師の判断が最優先されます。

リスクファクターの詳細な情報を提供することで、より個別化された鑑別診断が得られました。

## 例2: 症状分析プロンプトの使用

### シナリオ

28歳女性から「今朝から激しい頭痛がある」という電話相談を受けました。緊急受診の必要性を判断する必要があります。

### 使用したプロンプト

[症状分析プロンプト](../prompts/diagnosis/symptom-analysis.md)を使用しました。

### 入力内容

```
あなたは救急医療に精通した臨床医です。以下の症状について詳細な分析を行ってください。

【患者基本情報】
- 年齢・性別: 28歳 女性
- 主訴: 頭痛

【症状の詳細】
- 発症時期: 今朝起床時から
- 発症様式: 突然
- 症状の程度: 重度（今まで経験したことのない激しい痛み）
- 持続時間: 持続性
- 増悪因子: 頭を動かすと悪化
- 寛解因子: 特になし、市販の鎮痛薬も無効
- 随伴症状: 嘔気・嘔吐1回、項部硬直感、光がまぶしい
- 既往歴: 特記事項なし
- 最近の変化: 特になし

【依頼内容】
以下の観点から症状を分析してください：
（以下省略）
```

### AIからの出力例（要約）

AIは以下のような分析を提示しました。

**重症度評価**では、重症と判定されました。「今まで経験したことのない」突然の激しい頭痛は、重大な疾患を示唆する可能性があります。

**緊急性の判断**では、緊急度が「高」と評価されました。Red flags（危険な徴候）として、突然発症、最悪の頭痛、項部硬直感、嘔吐、光過敏が指摘されました。推奨される対応時間は「即座」とされました。

**推奨される対応**として、直ちに救急受診が必要と判断されました。くも膜下出血などの重大な疾患の可能性があり、頭部CT、必要に応じて腰椎穿刺などの精査が必要とされました。

### 臨床での活用

この分析に基づき、患者に直ちに救急外来を受診するよう指示しました。救急外来での頭部CTでくも膜下出血が確認され、緊急手術が行われました。

AIプロンプトは、電話相談でのトリアージ判断を支援し、緊急性の高い症状を見逃さないための助けになりました。

### 学んだポイント

「今まで経験したことのない」「突然の」といったRed flagsの表現を正確に捉え、入力することが重要です。

電話相談では視診や触診ができないため、口頭で確認できる情報（随伴症状、発症様式など）を詳しく聴取することが特に重要です。

緊急性の判断では、AIの分析を参考にしつつも、最終的には医療従事者の臨床判断が必要です。

## まとめ

診断支援プロンプトは、臨床判断を支援する有用なツールです。ただし、AIの出力はあくまで参考情報であり、最終的な医療判断は必ず有資格の医療従事者が行う必要があります。

詳細な情報を提供することで、より精度の高い分析が得られます。緊急性の高い症状では、AIの分析を待たずに適切な対応を優先してください。
