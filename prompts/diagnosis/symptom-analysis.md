# 症状分析プロンプト

## 概要

患者の症状を体系的に分析し、重症度評価、緊急性の判断、必要な対応を提案するプロンプトです。

## プロンプトテンプレート

```
あなたは救急医療に精通した臨床医です。以下の症状について詳細な分析を行ってください。

【患者基本情報】
- 年齢・性別: [年齢]歳 [性別]
- 主訴: [主訴]

【症状の詳細】
- 発症時期: [いつから]
- 発症様式: [突然 / 徐々に / その他]
- 症状の程度: [軽度 / 中等度 / 重度]
- 持続時間: [持続性 / 間欠性]
- 増悪因子: [症状が悪化する要因]
- 寛解因子: [症状が軽減する要因]
- 随伴症状: [他にある症状]
- 既往歴: [関連する既往歴]
- 最近の変化: [生活環境、ストレス、新規内服薬など]

【依頼内容】
以下の観点から症状を分析してください：

1. **症状の特徴づけ**
   - 症状のパターン分析
   - 典型的な疾患との一致度
   - 非典型的な要素の指摘

2. **重症度評価**
   - 現在の重症度（軽症/中等症/重症）
   - 評価の根拠
   - バイタルサインで確認すべき項目

3. **緊急性の判断**
   - 緊急度（低/中/高）
   - Red flags（危険な徴候）の有無
   - 推奨される対応時間（即座/数時間以内/24時間以内/数日以内）

4. **推奨される対応**
   - 救急受診の必要性
   - 外来受診で対応可能か
   - 自宅での経過観察の可否
   - 患者・家族への説明ポイント

5. **追加で確認すべき情報**
   - 診断に重要な追加の問診項目
   - 必要な身体診察
   - 優先すべき検査

注意: この分析は医療従事者の判断を支援するものであり、最終的な判断は実際に患者を診察した医療従事者が行ってください。
```

あなたは救急医療に精通した臨床医です。以下の症状について詳細な分析を行ってください。

- 年齢・性別: 28歳 女性
- 主訴: 頭痛

【症状の詳細】

## ベストプラクティス

### 効果的な使い方

**OPQRST法**を活用して症状を記載すると、より構造化された分析が得られます。Onset（発症様式）、Provocation/Palliation（増悪・寛解因子）、Quality（性質）、Region/Radiation（部位・放散）、Severity（重症度）、Time（時間経過）の観点で情報を整理しましょう。

**Red flagsの確認**は特に重要です。「今まで経験したことのない」「突然の」「急速に悪化する」といった表現は、緊急性の高い疾患を示唆する可能性があります。

**随伴症状**は診断の重要な手がかりです。主訴以外の症状も詳しく聴取し、記載してください。

### 注意点

**緊急性の高い症状**（突然の激しい頭痛、胸痛、呼吸困難、意識障害など）では、AIの分析を待たずに直ちに適切な医療機関への受診を促してください。

このプロンプトは**トリアージの補助**として使用できますが、最終的なトリアージ判断は医療従事者が行う必要があります。

**電話相談**などで使用する場合は、視診や触診ができないことを考慮し、より慎重な判断が求められます。

## カスタマイズ

診療場面に応じてカスタマイズできます。

**救急外来向け**: バイタルサインの情報を追加し、より詳細な緊急度評価を依頼

**電話相談向け**: 視覚的・触診的情報が得られない前提で、口頭で確認できる情報に焦点を当てる

**慢性症状向け**: 発症時期を「数週間前から」「数ヶ月前から」に変更し、生活への影響や経過を重視
