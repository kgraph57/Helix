# 検査結果解釈プロンプト

## 概要

臨床検査データを解釈し、異常値の臨床的意義、考えられる病態、追加検査の必要性を評価するプロンプトです。

## プロンプトテンプレート

```
あなたは臨床検査医学に精通した専門医です。以下の検査結果について、臨床的解釈を行ってください。

【患者情報】
- 年齢・性別: [年齢]歳 [性別]
- 主訴・臨床症状: [主訴と主な症状]
- 既往歴: [既往歴]
- 内服薬: [現在の内服薬]
- 検査目的: [スクリーニング / 症状精査 / 経過観察 / その他]

【検査結果】
[検査項目と結果を記載]

例:
■ 血算
- WBC: [値] /μL (基準値: 3,500-9,000)
- RBC: [値] ×10⁴/μL (基準値: 男性400-550, 女性380-500)
- Hb: [値] g/dL (基準値: 男性13.5-17.5, 女性11.5-15.0)
- Ht: [値] % (基準値: 男性40-50, 女性35-45)
- MCV: [値] fL (基準値: 80-100)
- MCH: [値] pg (基準値: 27-34)
- MCHC: [値] % (基準値: 31-36)
- Plt: [値] ×10⁴/μL (基準値: 15-35)

■ 生化学
- TP: [値] g/dL (基準値: 6.5-8.0)
- Alb: [値] g/dL (基準値: 3.8-5.2)
- T-Bil: [値] mg/dL (基準値: 0.2-1.2)
- AST: [値] U/L (基準値: 10-40)
- ALT: [値] U/L (基準値: 5-40)
- ALP: [値] U/L (基準値: 100-340)
- γ-GTP: [値] U/L (基準値: 男性<70, 女性<30)
- BUN: [値] mg/dL (基準値: 8-20)
- Cr: [値] mg/dL (基準値: 男性0.6-1.1, 女性0.4-0.8)
- eGFR: [値] mL/min/1.73m² (基準値: >60)
- Na: [値] mEq/L (基準値: 135-145)
- K: [値] mEq/L (基準値: 3.5-5.0)
- Cl: [値] mEq/L (基準値: 98-108)
- Ca: [値] mg/dL (基準値: 8.5-10.5)
- Glu: [値] mg/dL (基準値: 空腹時70-110)
- HbA1c: [値] % (基準値: <5.6)
- CRP: [値] mg/dL (基準値: <0.3)

【依頼内容】
以下の形式で検査結果を解釈してください：

1. **異常値のまとめ**
   - 高値を示している項目
   - 低値を示している項目
   - 正常範囲だが注目すべき項目

2. **臨床的解釈**
   - 各異常値の臨床的意義
   - 異常値のパターンから考えられる病態
   - 患者の症状や既往歴との関連性

3. **考えられる疾患・病態**
   - 検査所見から示唆される疾患
   - 優先度の高い順に列挙
   - 各疾患を支持する所見と矛盾する所見

4. **推奨される追加検査**
   - 診断確定のために必要な検査
   - 優先順位と検査の目的
   - 緊急性の評価

5. **フォローアップの推奨**
   - 再検査の必要性とタイミング
   - 経過観察のポイント
   - 専門医へのコンサルトの必要性

6. **患者への説明のポイント**
   - わかりやすい説明の例
   - 生活指導が必要な場合の内容

注意: この解釈は臨床判断の参考情報です。最終的な診断と治療方針は、患者を診察した医師が総合的に判断してください。
```

## ベストプラクティス

### 効果的な使い方

**基準値を併記**することで、AIがより正確に異常値を判断できます。検査施設によって基準値が異なる場合があるため、使用した検査施設の基準値を記載してください。

**臨床情報の提供**は検査結果の解釈に不可欠です。症状、既往歴、内服薬などの情報を詳しく記載することで、より臨床的に意義のある解釈が得られます。

**経時的変化**がある場合は、過去の検査結果も併記すると、トレンドの評価が可能になります。

### 注意点

**検査値の解釈は臨床文脈が重要**です。同じ異常値でも、患者の年齢、性別、症状、既往歴によって臨床的意義が大きく異なります。

**緊急性の高い異常値**（重度の電解質異常、著明な腎機能障害など）では、AIの解釈を待たずに直ちに適切な対応を行ってください。

**検査の限界**を理解することも重要です。正常値でも疾患が否定できない場合や、偽陽性・偽陰性の可能性も考慮してください。

## カスタマイズ

検査の種類や目的に応じてカスタマイズできます。

**専門検査向け**: 内分泌検査、腫瘍マーカー、自己抗体など、特殊検査の解釈を依頼

**画像検査向け**: CT、MRI、超音波などの画像所見の解釈を依頼（所見を詳細に記載）

**経過観察向け**: 複数時点の検査結果を提示し、トレンド分析を依頼
