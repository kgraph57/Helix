# 鑑別診断プロンプト

## 概要

患者の主訴、症状、既往歴、身体所見から、考えられる鑑別診断をリストアップし、それぞれの可能性を評価するプロンプトです。

## プロンプトテンプレート

```
あなたは経験豊富な臨床医です。以下の患者情報に基づいて、鑑別診断を行ってください。

【患者情報】
- 年齢・性別: [年齢]歳 [性別]
- 主訴: [主訴]
- 現病歴: [現病歴の詳細]
- 既往歴: [既往歴]
- 内服薬: [現在の内服薬]
- アレルギー: [薬剤・食物アレルギー]
- 家族歴: [家族歴]
- 生活歴: [喫煙・飲酒・職業など]
- バイタルサイン: 
  - 血圧: [血圧] mmHg
  - 脈拍: [脈拍] /分
  - 体温: [体温] ℃
  - 呼吸数: [呼吸数] /分
  - SpO2: [SpO2] %
- 身体所見: [身体所見の詳細]
- 検査所見: [既存の検査結果があれば]

【依頼内容】
以下の形式で鑑別診断を提示してください：

1. **最も可能性の高い診断（Top 3）**
   - 各診断名
   - その診断を支持する所見
   - 確定診断のために必要な追加検査

2. **除外すべき重要な疾患（Must not miss）**
   - 緊急性の高い疾患
   - 見逃すと重大な結果をもたらす疾患

3. **その他の鑑別診断**
   - 可能性は低いが考慮すべき診断

4. **推奨される初期対応**
   - 追加で行うべき検査
   - 初期治療の方針
   - 専門医へのコンサルトの必要性

注意: この鑑別診断はあくまで参考情報です。最終的な診断と治療方針は、実際に患者を診察した医師が決定してください。
```

## 使用例

### 入力例

```
あなたは経験豊富な臨床医です。以下の患者情報に基づいて、鑑別診断を行ってください。

【患者情報】
- 年齢・性別: 55歳 男性
- 主訴: 胸痛
- 現病歴: 3時間前から持続する胸部圧迫感。左肩への放散痛あり。冷汗、悪心を伴う。安静にしても改善しない。
- 既往歴: 高血圧（10年）、2型糖尿病（5年）、脂質異常症
- 内服薬: アムロジピン5mg、メトホルミン1000mg、アトルバスタチン10mg
- アレルギー: なし
- 家族歴: 父親が50代で心筋梗塞
- 生活歴: 喫煙20本/日×30年、飲酒機会飲酒、デスクワーク
- バイタルサイン: 
  - 血圧: 150/95 mmHg
  - 脈拍: 98 /分（整）
  - 体温: 36.8 ℃
  - 呼吸数: 20 /分
  - SpO2: 96 % (室内気)
- 身体所見: 冷汗あり、顔面蒼白、心音整、雑音なし、呼吸音清、腹部異常なし
- 検査所見: なし（救急外来初診）

以下の形式で鑑別診断を提示してください：

```

## ベストプラクティス

### 効果的な使い方

**詳細な情報提供**を心がけてください。症状の発症時期、経過、増悪・寛解因子など、できるだけ詳しい情報を入力することで、より精度の高い鑑別診断が得られます。

**バイタルサインと身体所見**は診断の重要な手がかりとなります。可能な限り具体的な数値や所見を記載してください。

**既往歴と家族歴**は疾患のリスク評価に重要です。関連する情報は漏れなく記載しましょう。

### 注意点

AIの出力は**あくまで参考情報**です。実際の診療では、患者を直接診察し、臨床判断を行った医師の判断が最優先されます。

緊急性の高い症状（胸痛、呼吸困難、意識障害など）の場合は、AIの出力を待たずに**適切な初期対応**を優先してください。

このプロンプトは**教育目的や診療支援**のためのツールであり、医師の臨床判断を代替するものではありません。

## カスタマイズ

このプロンプトは、診療科や状況に応じてカスタマイズできます。

**小児科向け**: 成長発達歴、予防接種歴、保育園・学校での流行状況などを追加

**産婦人科向け**: 月経歴、妊娠・出産歴、最終月経日などを追加

**精神科向け**: 精神症状の詳細、社会的背景、ストレス因子などを追加
