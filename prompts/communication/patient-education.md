# 患者教育プロンプト

## 概要

患者やその家族に対して、疾患や治療についてわかりやすく説明するための文章を作成するプロンプトです。医学的に正確でありながら、専門用語を避けた平易な表現を使用します。

## プロンプトテンプレート

```
あなたは患者教育に精通した医療従事者です。以下の内容について、患者とその家族が理解しやすい説明文を作成してください。

【対象患者】
- 年齢・性別: [年齢]歳 [性別]
- 診断名: [診断名]
- 教育レベル: [小学生レベル / 中学生レベル / 高校生レベル / 一般成人レベル]
- 健康リテラシー: [低い / 普通 / 高い]
- 特記事項: [視覚障害、聴覚障害、認知機能低下など、配慮が必要な点]

【説明したい内容】
[以下から選択、または具体的に記載]
- 疾患の基本的な説明
- 検査の目的と方法
- 治療の内容と効果
- 薬の飲み方と副作用
- 生活上の注意点
- 予後と今後の見通し
- その他: [具体的な内容]

【説明の目的】
- [理解を深める / 不安を軽減する / 治療への協力を得る / 生活習慣の改善を促す / その他]

【依頼内容】
以下の要件を満たす説明文を作成してください：

1. **導入部分**
   - 患者の状況への共感
   - これから説明する内容の概要

2. **主要な説明**
   - 疾患・治療の説明（専門用語は平易な言葉に置き換え）
   - 図や例えを使った理解しやすい説明
   - 重要なポイントの強調

3. **具体的な行動指針**
   - 患者が実際に行うべきこと
   - 避けるべきこと
   - 日常生活での具体的な注意点

4. **よくある質問への回答**
   - 患者が疑問に思いやすい点
   - その回答

5. **フォローアップ情報**
   - 次回受診のタイミング
   - 連絡が必要な症状（警告徴候）
   - 相談窓口の情報

6. **励ましのメッセージ**
   - 前向きな見通し
   - サポート体制の説明

【注意事項】
- 専門用語は使わず、使う場合は必ず平易な言葉で説明を加える
- 短い文章で、段落を分けて読みやすくする
- 否定的な表現は避け、前向きで希望を持てる表現を使う
- 患者の不安に寄り添う共感的な表現を含める
- 医学的に正確な情報を提供する
- 文化的・社会的背景への配慮を示す
```

## 使用例

### 入力例

```
あなたは患者教育に精通した医療従事者です。以下の内容について、患者とその家族が理解しやすい説明文を作成してください。

【対象患者】
- 年齢・性別: 55歳 男性
- 診断名: 2型糖尿病（新規診断）
- 教育レベル: 一般成人レベル
- 健康リテラシー: 普通
- 特記事項: なし

- 糖尿病の基本的な説明

- 生活習慣の改善方法

- 生活習慣の改善を促す
- 治療への積極的な参加を得る

以下の要件を満たす説明文を作成してください：
```

## ベストプラクティス

### 効果的な使い方

**患者の理解度に合わせた説明**が重要です。教育レベルや健康リテラシーの情報を提供することで、適切な難易度の説明文が得られます。

**具体的な行動指針**を含めることで、患者が実際に何をすればよいのかが明確になり、治療へのアドヒアランスが向上します。

**視覚的な要素**（図、イラスト、例え話）を活用することで、複雑な医学的概念もわかりやすく伝えられます。プロンプトで「図や例えを使って」と指定しましょう。

### 注意点

**専門用語の使用**は最小限にし、使用する場合は必ず平易な言葉で説明を加えてください。患者が理解できない言葉は不安を増大させます。

**否定的な表現や脅し**は避け、前向きで希望を持てる表現を使用してください。ただし、楽観的すぎる表現も避け、現実的な見通しを伝えることが重要です。

**文化的・社会的背景**への配慮も必要です。患者の価値観や信念を尊重した説明を心がけてください。

### カスタマイズ

対象や状況に応じてカスタマイズできます。

**小児向け**: 子どもの年齢に応じた言葉遣い、親への説明も含める

**高齢者向け**: 大きな文字、シンプルな構成、繰り返しの説明を含める

**外国人患者向け**: やさしい日本語、文化的配慮、通訳の活用を考慮

**書面配布用**: 印刷して配布できる形式、連絡先情報を明記
