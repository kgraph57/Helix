# 症例提示プロンプト

## 概要

臨床症例を教育的な形式で構造化して提示するプロンプトです。カンファレンスや学会発表、症例報告の作成に活用できます。

## プロンプトテンプレート

```
あなたは症例提示に精通した臨床医です。以下の症例情報を、教育的で構造化された症例提示の形式にまとめてください。

【症例の基本情報】
- 年齢・性別: [年齢]歳 [性別]
- 職業: [職業]
- 主訴: [主訴]
- 現病歴: [詳細な現病歴]
- 既往歴: [既往歴]
- 家族歴: [家族歴]
- 生活歴: [喫煙・飲酒・その他]
- 内服薬: [内服薬]
- アレルギー: [アレルギー]

【来院時所見】
- バイタルサイン: [血圧、脈拍、体温、呼吸数、SpO2]
- 身体所見: [詳細な身体所見]
- 神経学的所見: [該当する場合]

【検査所見】
- 血液検査: [結果]
- 尿検査: [結果]
- 画像検査: [X線、CT、MRI、超音波など]
- 心電図: [所見]
- その他の検査: [該当する検査]

【臨床経過】
- 初期診断と治療: [内容]
- 経過: [時系列での経過]
- 最終診断: [確定診断]
- 治療内容: [詳細な治療]
- 転帰: [予後]

【提示の目的】
[以下から選択]
- カンファレンスでの症例検討
- 学会発表用の症例報告
- 論文投稿用の症例報告
- 教育目的の症例提示
- その他: [具体的な目的]

【依頼内容】
以下の形式で症例を提示してください：

1. **症例サマリー（冒頭）**
   - 1-2文で症例の核心を要約
   - この症例の教育的価値

2. **症例提示**
   - 主訴と現病歴（時系列で整理）
   - 既往歴・家族歴・生活歴（関連性の高い情報を強調）
   - 来院時身体所見（陽性所見と重要な陰性所見）
   - 検査所見（異常値を強調、正常範囲も記載）

3. **臨床推論のポイント**
   - この時点での鑑別診断
   - 診断の手がかりとなった所見
   - 診断を進める上での考え方

4. **診断と治療**
   - 確定診断に至るプロセス
   - 診断の根拠
   - 治療方針の決定理由
   - 実際の治療内容

5. **臨床経過と転帰**
   - 治療への反応
   - 経過中の重要なイベント
   - 最終的な転帰

6. **考察**
   - この症例の特徴的な点
   - 診断・治療上の課題
   - 文献的考察
   - 学んだ教訓（Take-home message）

7. **まとめ（Learning Points）**
   - この症例から学ぶべき3-5つのポイント

【注意事項】
- 患者のプライバシーに配慮し、個人が特定できる情報は匿名化する
- 時系列を明確にし、因果関係がわかりやすいように記述する
- 重要な所見を強調し、診断の根拠を明確にする
- 教育的な観点から、一般化できる学びを抽出する
- 医学的に正確な用語を使用する
```

## 使用例

### 入力例

```
あなたは症例提示に精通した臨床医です。以下の症例情報を、教育的で構造化された症例提示の形式にまとめてください。

【症例の基本情報】
- 年齢・性別: 72歳 女性
- 職業: 無職（元教員）
- 主訴: 発熱、咳嗽、呼吸困難
- 現病歴: 5日前から発熱（38℃台）と咳嗽が出現。3日前から呼吸困難が増悪し、食事摂取も困難となったため救急外来を受診。
- 既往歴: 高血圧、脂質異常症、変形性膝関節症
- 家族歴: 特記事項なし
- 生活歴: 非喫煙、機会飲酒、独居（娘が近隣に居住）
- 内服薬: アムロジピン5mg、ロスバスタチン5mg
- アレルギー: なし

【来院時所見】
- バイタルサイン: BP 142/85 mmHg, HR 102/min, BT 38.5℃, RR 28/min, SpO2 88% (室内気)
- 身体所見: 意識清明、呼吸促迫あり、右下肺野でcoarse crackles聴取、心音整、腹部異常なし、下腿浮腫なし
- 神経学的所見: 異常なし

【検査所見】
- 血液検査: WBC 14,500/μL (Neut 85%), CRP 18.5 mg/dL, Cr 1.2 mg/dL, BUN 28 mg/dL
- 胸部X線: 右下肺野に浸潤影
- 血液ガス: pH 7.42, PaCO2 38 mmHg, PaO2 58 mmHg (室内気)

【臨床経過】
- 初期診断と治療: 市中肺炎と診断。A-DROP 3点（中等症）。セフトリアキソン2g/日開始、酸素投与開始。
- 経過: 入院3日目に解熱、呼吸状態改善。喀痰培養から肺炎球菌検出。抗菌薬は7日間継続。
- 最終診断: 肺炎球菌による市中肺炎
- 治療内容: セフトリアキソン7日間、酸素療法
- 転帰: 入院10日目に軽快退院

【提示の目的】
- 教育目的の症例提示（研修医向けカンファレンス）

【依頼内容】
以下の形式で症例を提示してください：
（以下同様）
```

## ベストプラクティス

### 効果的な使い方

**時系列を明確に**記載することで、診断・治療のプロセスが理解しやすくなります。発症から受診、診断、治療、転帰までの流れを整理しましょう。

**陽性所見と重要な陰性所見**の両方を記載してください。鑑別診断を進める上で、「ない」ことが重要な情報となる場合があります。

**教育的な視点**を意識してください。この症例から何を学ぶべきか、一般化できる教訓は何かを明確にすることが重要です。

### 注意点

**患者のプライバシー保護**は最優先です。年齢を幅で示す（「70代」など）、地域を特定できる情報を削除するなど、個人が特定できないように配慮してください。

**バイアスの回避**も重要です。後知恵バイアス（結果を知った上での解釈）を避け、当時の臨床判断のプロセスを正確に記述してください。

**文献的考察**を含める場合は、最新のエビデンスやガイドラインを参照してください。AIの出力は参考情報として、実際の文献を確認することが重要です。

## カスタマイズ

目的や対象に応じてカスタマイズできます。

**学会発表用**: 簡潔で視覚的な構成、スライドに適した箇条書き形式を依頼

**論文投稿用**: 雑誌の投稿規定に合わせた形式、文献引用の形式を指定

**教育カンファレンス用**: ディスカッションポイント、クイズ形式の要素を追加

**希少疾患の症例**: 文献レビュー、類似症例との比較を重点的に
