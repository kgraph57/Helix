# 治療計画立案プロンプト

## 概要

診断された疾患に対して、エビデンスに基づいた治療計画を立案し、複数の治療オプションを比較検討するプロンプトです。

## プロンプトテンプレート

```
あなたはエビデンスに基づいた医療(EBM)に精通した臨床医です。以下の患者に対する治療計画を立案してください。

【患者情報】
- 年齢・性別: [年齢]歳 [性別]
- 診断名: [確定診断名]
- 重症度: [軽症 / 中等症 / 重症]
- 病期・ステージ: [該当する場合]

【臨床情報】
- 主な症状: [現在の症状]
- 既往歴: [既往歴]
- 併存疾患: [合併症]
- 内服薬: [現在の内服薬]
- アレルギー: [薬剤アレルギー]
- 腎機能: Cr [値] mg/dL, eGFR [値] mL/min/1.73m²
- 肝機能: AST [値] U/L, ALT [値] U/L
- その他重要な検査所見: [関連する検査結果]

【患者背景】
- 社会的背景: [独居/家族同居、介護者の有無など]
- 職業・活動性: [職業、日常生活動作レベル]
- 治療に対する希望: [患者・家族の希望や価値観]
- 経済的考慮: [医療費の負担能力、保険の種類]

【依頼内容】
以下の形式で治療計画を提案してください：

1. **治療目標**
   - 短期目標（数日〜数週間）
   - 中期目標（数ヶ月）
   - 長期目標（数年）
   - QOL改善の目標

2. **推奨される治療オプション**
   各オプションについて以下を記載：
   - 治療法の概要
   - エビデンスレベル（ガイドライン推奨度）
   - 期待される効果
   - 主な副作用・リスク
   - 治療期間と費用の概算
   - この患者への適用可能性

3. **第一選択の治療計画**
   - 選択理由
   - 具体的な治療内容（薬剤名、用量、投与経路、期間）
   - 併用療法（必要な場合）
   - 非薬物療法（生活指導、リハビリなど）

4. **モニタリング計画**
   - 効果判定の指標と評価時期
   - 副作用モニタリングの項目と頻度
   - 必要な定期検査

5. **代替案・次善策**
   - 第一選択が使用できない場合の代替案
   - 治療抵抗性の場合の対応
   - 専門医へのコンサルトのタイミング

6. **患者教育のポイント**
   - 疾患と治療についての説明
   - 服薬アドヒアランス向上のための工夫
   - 生活上の注意点
   - 受診が必要な症状（警告徴候）

7. **多職種連携**
   - 必要な他職種（薬剤師、看護師、理学療法士など）
   - 連携のポイント

注意: この治療計画は最新のガイドラインとエビデンスに基づいていますが、最終的な治療方針は患者を診察した医師が、患者の個別性を考慮して決定してください。
```

## ベストプラクティス

### 効果的な使い方

**患者の個別性を重視**してください。年齢、腎機能、肝機能、併存疾患などの情報を詳しく提供することで、より個別化された治療計画が得られます。

**患者の価値観や希望**を記載することで、shared decision making（共同意思決定）に基づいた治療計画の立案が可能になります。

**複数の疾患がある場合**は、疾患間の相互作用や薬剤の相互作用も考慮した総合的な治療計画を依頼できます。

### 注意点

**ガイドラインは参考**であり、個々の患者の状況に応じた柔軟な対応が必要です。特に高齢者や複数の併存疾患がある患者では、標準的な治療が適さない場合があります。

**薬剤の用量や投与方法**は、必ず最新の添付文書やガイドラインで確認してください。AIの出力に誤りがある可能性も考慮してください。

**治療の優先順位**を考慮することも重要です。複数の疾患がある場合、どの疾患の治療を優先すべきか、臨床判断が必要です。

## カスタマイズ

疾患や状況に応じてカスタマイズできます。

**がん治療向け**: 病期、パフォーマンスステータス、遺伝子変異情報などを追加し、集学的治療計画を依頼

**慢性疾患向け**: 長期的な管理計画、合併症予防、QOL維持に焦点を当てる

**急性期治療向け**: 緊急性、入院の必要性、初期安定化の方法に焦点を当てる
