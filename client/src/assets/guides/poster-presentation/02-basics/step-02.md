# ステップ2: 情報の優先順位付け

## このステップの目的

重要な情報を強調し、補足情報は控えめにします。ポスターは限られたスペースのため、情報の取捨選択が重要です。

## 優先順位の原則

### 最優先（大きく、目立つ）

- **タイトル**: 研究の内容が一目で分かる
- **主要な結果**: 最も重要な発見
- **結論**: 研究の要点

### 中優先（中程度のサイズ）

- **背景**: 研究の必要性
- **方法**: 研究の概要
- **図表**: 結果の可視化

### 低優先（小さく、控えめ）

- **詳細な方法**: 補足情報
- **参考文献**: 最小限
- **謝辞**: 控えめに

## 情報の簡潔化

### テキストの削減

- 論文の本文をそのまま使用しない
- 箇条書きを活用
- 重要なポイントのみを記載

### 図表の活用

- テキストより図表で伝える
- 1つの図表で複数の情報を伝える
- カラーを効果的に使用

## AIツールを活用した情報整理

**ChatGPT / Claude プロンプト例:**
```
以下の研究内容を、ポスター用に簡潔に整理してください。

研究内容:
[論文の本文または詳細な内容]

以下の要件を満たすように整理してください：
1. 各セクションを箇条書きで簡潔に
2. 重要な情報を強調
3. 補足情報は最小限に
4. ポスターで伝えるべき要点を明確に
```

## よくある質問

**Q: どのくらいの情報量が適切ですか？**
A: 通常、ポスター全体で500-800語程度です。多すぎると読みにくくなります。

**Q: 詳細な方法は省略してもよいですか？**
A: 主要な方法は記載し、詳細は省略または補足資料として準備します。
