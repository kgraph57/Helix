# ステップ1: レイアウト設計

## このステップの目的

効果的な情報配置を設計します。ポスターは限られたスペースで情報を伝える必要があるため、レイアウトが重要です。

## ポスターのサイズ

### 一般的なサイズ

- **A0**: 841 × 1189 mm（最も一般的）
- **A1**: 594 × 841 mm
- **カスタムサイズ**: 学会の指定に従う

## レイアウトの原則

### 1. 情報の流れ

- **左から右、上から下**: 自然な読みの流れ
- **明確な順序**: 背景 → 方法 → 結果 → 結論

### 2. セクションの配置

**推奨レイアウト:**
```
[タイトル・著者]
[背景] [方法]
[結果] [結論]
[参考文献] [謝辞]
```

### 3. 視覚的階層

- **タイトル**: 最も大きく、目立つ
- **セクション見出し**: 中程度のサイズ
- **本文**: 読みやすいサイズ（通常18-24pt以上）

## AIツールを活用したレイアウト設計

**ChatGPT / Claude プロンプト例:**
```
以下の研究内容のポスターのレイアウトを提案してください。

研究内容:
- タイトル: [タイトル]
- 背景: [背景]
- 方法: [方法]
- 結果: [結果]
- 結論: [結論]

ポスターサイズ: A0（841 × 1189 mm）

以下の情報を提供してください：
1. 推奨レイアウト（セクションの配置）
2. 各セクションの推奨サイズ
3. 視覚的階層の設計
4. デザインのポイント
```

## よくある質問

**Q: どのレイアウトが最も効果的ですか？**
A: 研究の内容によりますが、一般的には左から右、上から下の流れが推奨されます。

**Q: セクションはいくつが適切ですか？**
A: 通常4-6セクションです。多すぎると情報が散漫になります。
