# ステップ5: 最終確認と提出

## このステップの目的

ポスターを最終確認し、提出準備を完了します。

## 最終確認のチェックリスト

### 内容の確認

- [ ] 情報が簡潔で要点が明確
- [ ] 図表が適切に配置されている
- [ ] 誤字脱字がない
- [ ] 専門用語が適切に使用されている

### デザインの確認

- [ ] 文字サイズが適切
- [ ] コントラストが十分
- [ ] 配色が適切
- [ ] レイアウトが読みやすい

### 形式の確認

- [ ] サイズが要件に準拠
- [ ] 解像度が適切（通常300dpi以上）
- [ ] ファイル形式が適切（PDF推奨）

## 印刷前の確認

### 1. プレビューの確認

実際のサイズでプレビューし、確認します。

- 文字が読めるか
- 図表が適切に表示されているか
- レイアウトが適切か

### 2. 印刷テスト

可能であれば、小さなサイズで印刷テストを実施します。

- 色の再現性
- 文字の読みやすさ

## AIツールを活用した最終確認

**ChatGPT / Claude プロンプト例:**
```
以下のポスターの内容を最終確認してください。

ポスターの内容:
[ポスターの主要な内容]

以下の観点で確認してください：
1. 内容の正確性
2. 情報の簡潔性
3. 専門用語の適切な使用
4. 改善すべき点

改善点があれば具体的に提案してください。
```

## よくある質問

**Q: 印刷はどこで依頼すればよいですか？**
A: 学会が指定する印刷業者、または大学の印刷サービスを利用します。

**Q: 提出期限はいつですか？**
A: 学会によって異なります。事前に確認します。

## まとめ

ポスター発表の作成は、研究を効果的に伝える上で重要です。適切なレイアウト、視認性の高いデザイン、簡潔な情報、質疑応答対策により、成功するポスター発表を作成できます。AIツールを適切に活用することで、効率化しながら高品質なポスターを作成できます。
