# ポスター発表作成ガイド

## このガイドについて

学会でのポスター発表は、研究成果を効果的に伝える重要な機会です。本ガイドでは、視認性が高く、情報が整理されたポスターの作成方法を解説します。

## 対象読者

- 学会でポスター発表を予定している研究者
- 初めてポスターを作成する方
- ポスターの質を向上させたい方

## このガイドで学べること

1. **レイアウト設計**: 効果的な情報配置
2. **情報の優先順位**: 重要な情報の強調
3. **視認性の向上**: 読みやすいデザイン
4. **質疑応答対策**: 想定問答集の作成
5. **AIツールの活用**: ポスター作成を効率化する方法

## 所要時間

- **全体**: 15-20分（読了時間）
- **実際の作業**: 数日〜1週間（ポスターの内容による）

## 前提知識

- 基本的なプレゼンテーションの知識
- 研究内容の理解

## 使用するツール

- **PowerPoint / Keynote**: ポスター作成
- **Canva / Adobe Illustrator**: デザイン
- **AIツール**: コンテンツ作成、デザイン提案
