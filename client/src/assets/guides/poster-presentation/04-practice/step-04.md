# ステップ4: 質疑応答対策

## このステップの目的

ポスター発表での質疑応答に備えます。想定される質問を準備し、明確に回答できるようにします。

## 想定される質問

### 一般的な質問

- **研究の背景**: なぜこの研究を実施したか
- **方法の詳細**: 研究デザイン、データ収集方法
- **結果の解釈**: 結果の意味、臨床的意義
- **限界**: 研究の限界と今後の課題

### 専門的な質問

- **統計解析**: 使用した統計手法、p値の解釈
- **交絡因子**: 交絡因子の調整方法
- **一般化可能性**: 結果の一般化可能性

## 回答の準備

### 簡潔に

- 1-2分で回答できる長さ
- 要点を明確に
- 補足資料を準備（必要に応じて）

### 明確に

- 専門用語を適切に使用
- 数値やデータで裏付ける
- 不確実な点は正直に述べる

## AIツールを活用した想定問答集作成

**ChatGPT / Claude プロンプト例:**
```
以下の研究のポスター発表用の想定問答集を作成してください。

研究内容:
- タイトル: [タイトル]
- 背景: [背景]
- 方法: [方法]
- 結果: [結果]
- 結論: [結論]

以下の情報を提供してください：
1. 想定される質問（10-15個）
2. 各質問への回答（簡潔に、1-2分で説明できる長さ）
3. 補足資料の提案（必要に応じて）
```

## よくある質問

**Q: 答えられない質問があった場合はどうすればよいですか？**
A: 正直に「現時点では回答できませんが、今後の課題として検討します」と答えます。

**Q: 質問の時間はどのくらいですか？**
A: 通常5-10分程度です。重要な質問に時間を割きます。

## まとめ

ポスター発表の作成は、レイアウト設計、情報の優先順位付け、視認性の向上、質疑応答対策など、複数の要素で構成されます。各要素を適切に設計することで、効果的なポスター発表ができます。
