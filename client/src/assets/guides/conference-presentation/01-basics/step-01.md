# ステップ1: カンファレンス発表の標準的な構成を理解する

## 症例カンファレンスの基本構成

質の高い症例発表には、標準的な構成があります。
この構成に沿って資料を作成することで、聴衆が理解しやすく、議論が活発になります。

### 標準的なスライド構成（10-15分発表）

標準的な症例発表は、約10スライド、15分で構成されます。

スライド1はタイトルスライドで、所要時間は30秒です。
症例の概要を一言で伝えることがポイントです。
スライド2は症例サマリーで、所要時間は1分です。
年齢、性別、主訴、診断を含めることがポイントです。
スライド3は現病歴で、所要時間は2分です。
時系列で簡潔に伝えることがポイントです。
スライド4は身体所見で、所要時間は1分です。
重要な所見のみを伝えることがポイントです。
スライド5は検査所見で、所要時間は2分です。
異常値をハイライトすることがポイントです。
スライド6は鑑別診断で、所要時間は2分です。
優先順位付きで伝えることがポイントです。
スライド7は診断根拠で、所要時間は2分です。
なぜこの診断に至ったかを説明することがポイントです。
スライド8は治療方針で、所要時間は2分です。
選択した理由を説明することがポイントです。
スライド9は経過で、所要時間は1分です。
治療後の変化を示すことがポイントです。
スライド10は考察・学び（Take-home message）で、所要時間は2分です。

合計で約10スライド、15分の構成となります。

---

## 各スライドの詳細

### 1. タイトルスライド

目的は、症例の核心を一言で伝えることです。

含めるべき情報として、症例の特徴（年齢・性別）があります。
主訴または診断を含めます。
発表者名・所属を含めます。
発表日を含めます。

良い例:
```
80歳女性、急性腹症で来院
診断: 上腸間膜動脈血栓症

発表者: 山田太郎（内科研修医）
2025年12月13日
```

悪い例:
```
症例発表

山田太郎
```

---

### 2. 症例サマリー

目的は、症例の全体像を1分で理解させることです。

含めるべき情報として、年齢・性別があります。
主訴を含めます。
現病歴を1-2文で簡潔に含めます。
診断を含めます。
転帰を含めます。

テンプレート:
```
【症例サマリー】
- 年齢・性別: 80歳、女性
- 主訴: 急性腹痛
- 現病歴: 来院3時間前から突然の腹痛が出現。嘔吐を伴う。
- 診断: 上腸間膜動脈血栓症
- 転帰: 緊急手術施行、術後経過良好
```

---

### 3. 現病歴

目的は、症状の経過を時系列で理解させることです。

時系列で整理し、発症時刻を明確にします。
重要な症状のみを伝え、詳細すぎないようにします。
関連する既往歴を必要に応じて含めます。

良い例:
```
【現病歴】
- X日 15:00: 突然の腹痛が出現（心窩部→全腹部）
- X日 16:00: 嘔吐2回、下痢なし
- X日 18:00: 症状増悪し救急搬送

【既往歴】
- 心房細動（ワルファリン内服中断中）
- 高血圧、脂質異常症
```

---

### 4. 身体所見

目的は、診断に重要な所見を強調することです。

バイタルサインを必ず記載します。
陽性所見を強調します。
陰性所見も重要で、除外診断に役立ちます。

テンプレート:
```
【バイタルサイン】
- 体温: 37.8°C
- 血圧: 100/60 mmHg
- 脈拍: 110回/分（不整）
- SpO2: 96%（室内気）

【腹部所見】
- 全腹部に圧痛（+）、筋性防御（-）
- 腸蠕動音: 減弱
- 直腸診: 血便（-）
```

---

### 5. 検査所見

目的は、診断に必要な検査結果を提示することです。

異常値をハイライトします。
太字や色付けで強調します。
正常値も記載し、基準範囲を示します。
画像所見は別スライドで提示します。

良い例:
```
【血液検査】
- WBC: 15,000 /μL ↑
- CRP: 8.5 mg/dL ↑
- Lactate: 4.2 mmol/L ↑
- D-dimer: 5.8 μg/mL ↑

【CT所見】
- 上腸間膜動脈に血栓（+）
- 小腸壁の造影不良
- 腹水（+）
```

---

### 6. 鑑別診断

目的は、診断プロセスを明示することです。

優先順位付きで、頻度や重症度を考慮します。
除外した理由も記載します。
3-5個に絞って提示します。

テンプレート:
```
【鑑別診断】
1. 上腸間膜動脈血栓症 ★
   - 心房細動、急性腹痛、Lactate上昇
2. 腸閉塞
   - 腸蠕動音減弱、嘔吐
   → CT で血栓を確認し除外
3. 急性膵炎
   - 腹痛、嘔吐
   → アミラーゼ正常で除外
```

---

### 7. 診断根拠

目的は、なぜこの診断に至ったかを説明することです。

臨床所見と検査所見を統合します。
診断基準があれば提示します。
確定診断の方法を説明します。

例:
```
【診断根拠】
1. 臨床所見
   - 心房細動（血栓リスク）
   - 急性腹痛（腹部所見に乏しい）
   
2. 検査所見
   - Lactate上昇（腸管虚血）
   - CT: 上腸間膜動脈血栓

3. 確定診断
   - 造影CT で上腸間膜動脈血栓を確認
```

---

### 8. 治療方針

目的は、選択した治療とその理由を説明することです。

治療選択肢を提示します。
選択した理由を明確にします。
代替案も考慮します。

テンプレート:
```
【治療選択肢】
1. 保存的治療（抗凝固療法）
2. 血管内治療（血栓溶解療法）
3. 外科的治療（血栓除去術）

【選択した治療】
- 緊急開腹術（血栓除去 + 腸管切除）

【理由】
- 腸管壊死の可能性が高い
- 発症から時間が経過
- 保存的治療では改善困難
```

---

### 9. 経過

目的は、治療後の変化を示すことです。

時系列で整理します。
改善点を強調します。
合併症があれば記載します。

例:
```
【術後経過】
- 術後1日目: ICU管理、バイタル安定
- 術後3日目: 一般病棟へ転棟
- 術後7日目: 経口摂取開始
- 術後14日目: 退院

【合併症】
- なし
```

---

### 10. 考察・学び（Take-home message）

目的は、この症例から何を学んだかを明確にすることです。

1-3個の重要なポイントを提示します。
一般化できる知識を伝えます。
今後の診療に活かせることを示します。

テンプレート:
```
【Take-home message】

1. 心房細動患者の急性腹痛では、
   上腸間膜動脈血栓症を疑う

2. 腹部所見に乏しくても、
   Lactate上昇があれば腸管虚血を考慮

3. 早期診断・早期治療が予後を左右する
```

見落としを防げます。
時間内に収まります。
