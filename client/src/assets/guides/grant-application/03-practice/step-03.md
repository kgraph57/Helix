# ステップ3: 予算計画の作成

## このステップの目的

適切な予算を算出し、根拠を明確にします。予算が過大または過小だと、申請が却下される可能性があります。

## 予算の種類

### 人件費

- 研究員の給与
- 謝金（外部専門家など）

### 物品費

- 消耗品
- 機器・備品
- ソフトウェア

### 旅費

- 国内旅費
- 海外旅費
- 宿泊費

### その他

- 印刷・製本費
- 通信費
- その他経費

## 予算の根拠

### 各経費の根拠を明確に

- **人件費**: 必要人数、期間、単価
- **物品費**: 必要数量、単価、見積もり
- **旅費**: 目的、回数、見積もり

## AIツールを活用した予算計画作成

**ChatGPT / Claude プロンプト例:**
```
以下の研究の予算計画を作成してください。

研究期間: 3年
研究内容: [研究内容]
必要なリソース:
- 研究員: 1名（3年間）
- データ収集: 500名分
- 統計解析ソフト: 1ライセンス
- 学会発表: 年2回（国内1回、国際1回）

以下の情報を提供してください：
1. 各経費の内訳
2. 各経費の根拠
3. 総額
4. 予算の妥当性の評価
```

## よくある質問

**Q: 予算はどのくらいが適切ですか？**
A: 研究の規模と内容に応じて適切に算出します。過大または過小にならないよう注意します。

**Q: 見積もりは必要ですか？**
A: 高額な物品費や旅費については、見積もりを添付することが推奨されます。
