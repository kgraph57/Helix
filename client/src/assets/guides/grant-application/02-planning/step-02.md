# ステップ2: 研究計画書の作成

## このステップの目的

明確で説得力のある研究計画書を作成します。研究計画書は、申請書の核心部分です。

## 研究計画書の構成

### 1. 研究の背景と意義

- **現状の問題点**: なぜこの研究が必要か
- **研究の重要性**: 研究の意義とインパクト
- **最新の研究動向**: 関連研究のレビュー

### 2. 研究目的

- **主要目的**: 研究の主要な目的
- **副次目的**: 副次的な目的（あれば）

### 3. 研究デザイン

- **研究の種類**: 観察研究、介入研究など
- **対象者**: 研究対象の定義
- **方法**: データ収集と解析方法

### 4. 研究の新規性と独創性

- **既存研究との違い**: 何が新しいか
- **独創性**: 研究の独創的な点

## AIツールを活用した研究計画書の作成

### 背景と意義の作成

**ChatGPT / Claude プロンプト例:**
```
以下の研究課題の背景と意義を、グラント申請書用に作成してください。

研究課題: [研究課題名]

以下の構成で作成してください：
1. 現状の問題点
2. 研究の重要性
3. 最新の研究動向
4. 研究の意義とインパクト

説得力があり、審査員が理解しやすい文章で作成してください。
```

### 研究目的の明確化

**プロンプト例:**
```
以下の研究課題の目的を、グラント申請書用に明確に定義してください。

研究課題: [研究課題名]
研究の背景: [研究の背景]

以下の形式で作成してください：
- 主要目的: [主要な目的]
- 副次目的: [副次的な目的（あれば）]

目的は測定可能で、達成可能な形で定義してください。
```

### 研究デザインの作成

**プロンプト例:**
```
以下の研究目的に基づいて、グラント申請書用の研究デザインを作成してください。

研究目的: [研究目的]
対象者: [対象者の定義]

以下の情報を含めてください：
1. 研究の種類（観察研究/介入研究など）
2. 研究デザインの詳細
3. データ収集方法
4. 統計解析計画
5. 研究の新規性と独創性

グラント申請書に適した形式で作成してください。
```

## よくある質問

**Q: 研究計画はどのくらい詳細に書くべきですか？**
A: 実施可能であることを示す程度の詳細さが必要です。過度に詳細すぎると柔軟性が失われます。

**Q: 新規性をどのように強調すればよいですか？**
A: 既存研究との明確な違いを示し、研究の独創性を具体的に説明します。
