# ステップ2: 研究計画書の作成

## このステップの目的

明確で説得力のある研究計画書を作成します。研究計画書は申請書の核心部分です。

## 研究計画書の要素

### 1. 研究の背景と根拠

- 研究の必要性
- 既存研究のレビュー
- 研究のギャップ

### 2. 研究の目的と仮説

- 明確な研究目的
- 検証する仮説
- 研究課題（PICO形式）

### 3. 研究デザインと方法

- 研究デザインの選択と理由
- 対象者の選定基準
- データ収集方法
- 統計解析計画

### 4. 実施計画

- 研究のスケジュール
- 各フェーズの内容
- マイルストーン

## AIツールを活用した研究計画書作成

**ChatGPT / Claude プロンプト例:**
```
以下の情報から、グラント申請書の研究計画書セクションを作成してください。

研究課題: [研究課題]
研究の背景: [背景]
研究目的: [目的]
研究デザイン: [デザイン]
対象者: [対象者]
データ収集: [データ収集方法]
統計解析: [統計解析計画]

以下の要件を満たす研究計画書を作成してください：
1. 明確で説得力のある内容
2. 実施可能性が示されている
3. 新規性と意義が明確
4. 専門的だが理解しやすい
```

## よくある質問

**Q: 研究計画はどのくらい詳細にすべきですか？**
A: 審査員が研究の実施可能性を判断できる程度に詳細にします。ただし、過度に詳細すぎると柔軟性が失われます。

**Q: 仮説は必須ですか？**
A: 探索的研究では仮説が明確でない場合もありますが、検証的研究では仮説を明確にします。
