# ステップ5: 期待される成果の説明

## このステップの目的

研究の意義とインパクトを明確に説明します。期待される成果は、研究費を獲得する上で重要な要素です。

## 期待される成果の種類

### 学術的成果

- 論文発表
- 学会発表
- データベースの構築

### 社会的インパクト

- 臨床診療への貢献
- 政策への影響
- 患者への利益

### 今後の展開

- 継続研究の計画
- 他分野への応用
- 国際的な展開

## 成果の説明方法

### 具体的に

- 抽象的な表現を避ける
- 数値目標を設定（可能な場合）
- タイムラインを明確に

### 説得力を持たせる

- 既存研究との比較
- 研究の新規性
- 実現可能性

## AIツールを活用した成果説明の作成

**ChatGPT / Claude プロンプト例:**
```
以下の研究の期待される成果を説明してください。

研究課題: [研究課題]
研究の目的: [目的]
研究の新規性: [新規性]

以下の情報を含む成果説明を作成してください：
1. 学術的成果（論文、学会発表など）
2. 社会的インパクト（臨床への貢献など）
3. 今後の展開（継続研究など）
4. 研究の意義とインパクト

説得力があり、具体的な内容にしてください。
```

## よくある質問

**Q: 成果を過大に表現してもよいですか？**
A: 現実的で実現可能な成果を記載します。過大な表現は信頼性を損ないます。

**Q: 数値目標は必要ですか？**
A: 可能な限り数値目標を設定します（例：論文3本、学会発表2回など）。

## まとめ

グラント申請書の作成は、研究計画書、予算計画、研究体制、期待される成果など、複数の要素で構成されます。各要素を明確で説得力のある内容にすることで、研究費を獲得できる可能性が高まります。
