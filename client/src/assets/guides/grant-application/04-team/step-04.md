# ステップ4: 研究体制の構築

## このステップの目的

効果的な研究体制を構築し、申請書に記載します。適切な研究体制は、研究の成功に不可欠です。

## 研究体制の要素

### 研究代表者

研究の責任者です。

- **役割**: 研究全体の統括
- **資格**: 研究を実施できる資格と経験

### 研究分担者

研究に参加する研究者です。

- **役割**: 各研究者の専門性を活かした分担
- **専門性**: 統計、臨床、基礎研究など

### 研究協力者

研究に協力する外部の専門家です。

- **役割**: 特定の専門分野での協力
- **協力内容**: データ提供、解析支援など

## 研究体制の設計

### 1. 必要な専門性の特定

研究計画に基づいて、必要な専門性を特定します。

- 臨床専門家
- 統計専門家
- 基礎研究専門家

### 2. メンバーの選定

必要な専門性を持つメンバーを選定します。

- 研究経験
- 専門性
- 協力体制

### 3. 役割分担の明確化

各メンバーの役割を明確にします。

- 研究代表者の役割
- 研究分担者の役割
- 研究協力者の役割

## AIツールを活用した研究体制の構築

**ChatGPT / Claude プロンプト例:**
```

研究計画:
- 研究の種類: [研究の種類]
- 必要な専門性: [必要な専門性]
- 研究規模: [研究規模]

以下の情報を含めてください：
1. 研究代表者の役割と資格
2. 研究分担者の役割と専門性
3. 研究協力者の役割と協力内容
4. 研究体制の図（組織図）

効果的で説得力のある研究体制を設計してください。
```

## よくある質問

**Q: 研究分担者は何人必要ですか？**
A: 研究の規模と必要な専門性に応じて決まります。過度に多いと管理が困難になります。

**Q: 外部の専門家を研究協力者として含めるべきですか？**
A: 研究に必要な専門性が内部にない場合は、外部の専門家を含めることが推奨されます。
