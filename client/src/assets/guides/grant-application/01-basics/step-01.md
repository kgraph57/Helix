# ステップ1: 申請書の構成理解

## このステップの目的

グラント申請書の基本的な構成を理解します。申請する研究費の種類によって構成が異なりますが、基本的な要素は共通しています。

## 一般的な構成要素

### 1. 研究課題名

- 簡潔で明確
- 研究の内容が分かる
- キーワードを含む

### 2. 研究の概要

- 研究の背景と目的
- 研究の新規性と意義
- 期待される成果

### 3. 研究計画

- 研究の目的と仮説
- 研究デザインと方法
- データ収集と解析計画

### 4. 研究体制

- 研究代表者と分担者
- 各メンバーの役割
- 研究機関の環境

### 5. 予算計画

- 必要な経費の内訳
- 各経費の根拠
- 研究期間

### 6. 期待される成果

- 学術的成果
- 社会的インパクト
- 今後の展開

## AIツールを活用した構成の確認

**ChatGPT / Claude プロンプト例:**
```
以下の研究課題のグラント申請書の構成を確認してください。

研究課題: [あなたの研究課題]
申請する研究費: [研究費の種類]

以下の情報を提供してください：
1. 必要なセクション
2. 各セクションの推奨される長さ
3. 各セクションで強調すべきポイント
4. よくあるミスと対策
```

## よくある質問

**Q: 申請書の長さはどのくらいが適切ですか？**
A: 研究費の種類によりますが、通常5-20ページ程度です。指定されたページ数を確認します。

**Q: どのセクションが最も重要ですか？**
A: 研究計画と期待される成果が最も重要です。明確で説得力のある内容にします。
