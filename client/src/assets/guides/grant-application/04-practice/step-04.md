# ステップ4: 研究体制の構築

## このステップの目的

効果的な研究体制を構築し、申請書に記載します。適切な研究体制は、研究の実施可能性を示す上で重要です。

## 研究体制の要素

### 研究代表者

- 研究の責任者
- 研究経験と実績
- 研究機関での立場

### 分担者

- 各専門分野の専門家
- 統計学者、臨床医、研究者など
- 各メンバーの役割

### 研究機関

- 研究環境（施設、設備）
- データへのアクセス
- 研究支援体制

## 効果的な研究体制

### 多様性

- 異なる専門分野の専門家
- 異なる研究機関からの参加
- 国際的な協力（可能な場合）

### 実績

- 各メンバーの研究実績
- 過去の共同研究の実績
- 研究機関の実績

## AIツールを活用した研究体制の検討

**ChatGPT / Claude プロンプト例:**
```
以下の研究の効果的な研究体制を提案してください。

研究課題: [研究課題]
必要な専門知識:
- 臨床医学
- 統計学
- データサイエンス

以下の情報を提供してください：
1. 必要なメンバーの種類
2. 各メンバーの役割
3. 研究体制の構成案
4. 各メンバーに期待される貢献
```

## よくある質問

**Q: 研究代表者は1名で十分ですか？**
A: 研究の規模によりますが、大規模研究では複数の代表者を置く場合もあります。

**Q: 外部の専門家を巻き込むべきですか？**
A: 必要な専門知識が内部にない場合は、外部の専門家を巻き込むことが推奨されます。
