# ステップ3: 予算計画の作成

## このステップの目的

適切で説得力のある予算計画を作成します。予算計画は、研究を実施するために必要な経費を明確に示します。

## 予算の種類

### 直接経費

研究に直接必要な経費です。

- **人件費**: 研究員の給与
- **旅費**: 学会参加、データ収集のための旅費
- **物品費**: 研究に必要な物品の購入
- **その他**: 謝金、印刷製本費など

### 間接経費

研究に間接的に必要な経費です。

- **間接経費**: 施設利用料など（グラントによって異なる）

## 予算計画の作成

### 1. 必要な経費の洗い出し

研究計画に基づいて、必要な経費を洗い出します。

- 人件費: 研究員の人数と期間
- 旅費: 学会参加、データ収集の回数
- 物品費: 必要な物品のリスト

### 2. 見積もりの取得

可能な限り正確な見積もりを取得します。

- 物品の価格: カタログや見積もり
- 旅費: 交通費、宿泊費の見積もり

### 3. 予算の根拠

各経費の根拠を明確にします。

- なぜその経費が必要か
- 金額の根拠

## AIツールを活用した予算計画の作成

**ChatGPT / Claude プロンプト例:**
```
以下の研究計画に基づいて、グラント申請書用の予算計画を作成してください。

研究計画:
- 研究期間: [期間]
- 研究員: [人数と期間]
- 学会参加: [回数]
- データ収集: [場所と回数]
- 必要な物品: [物品のリスト]

以下の形式で予算計画を作成してください：
1. 人件費: [金額と根拠]
2. 旅費: [金額と根拠]
3. 物品費: [金額と根拠]
4. その他: [金額と根拠]
5. 合計: [総額]

各経費の根拠を明確に記載してください。
```

## よくある質問

**Q: 予算はどのくらい申請すればよいですか？**
A: 研究を実施するために必要な最小限の金額を申請します。過度に多いと却下される可能性があります。

**Q: 予算の根拠はどのくらい詳細に書くべきですか？**
A: 審査員が理解できる程度の詳細さが必要です。各経費の必要性と金額の根拠を明確にします。
