# ステップ5: 期待される成果の明確化

## このステップの目的

研究により期待される成果を明確にし、研究の意義とインパクトを示します。

## 期待される成果の種類

### 学術的成果

- **論文発表**: 学術論文の発表
- **学会発表**: 学会での発表
- **データの提供**: 他の研究者へのデータ提供

### 社会的成果

- **臨床への応用**: 臨床診療への応用
- **政策への貢献**: 医療政策への貢献
- **患者への利益**: 患者への直接的な利益

### 長期的成果

- **研究の発展**: 今後の研究への発展
- **人材育成**: 若手研究者の育成

## 成果の明確化

### 1. 具体的な成果

抽象的ではなく、具体的な成果を示します。

- 論文の数とジャーナル
- 学会発表の数
- 臨床への応用方法

### 2. 成果の測定

成果を測定可能な形で示します。

- 論文のインパクトファクター
- 臨床での使用状況
- 患者への影響

### 3. 成果の意義

成果の意義とインパクトを明確にします。

- 学術的な意義
- 社会的な意義
- 長期的な影響

## AIツールを活用した成果の明確化

**ChatGPT / Claude プロンプト例:**
```
以下の研究計画に基づいて、グラント申請書用の期待される成果を明確化してください。

研究計画:
- 研究の目的: [研究の目的]
- 研究の種類: [研究の種類]
- 研究期間: [研究期間]

以下の情報を含めてください：
1. 学術的成果（論文、学会発表など）
2. 社会的成果（臨床への応用、政策への貢献など）
3. 長期的成果（研究の発展、人材育成など）
4. 成果の測定方法
5. 成果の意義とインパクト

具体的で測定可能な成果を示してください。
```

## よくある質問

**Q: 成果はどのくらい具体的に書くべきですか？**
A: 可能な限り具体的に書きます。論文の数やジャーナルのレベルなど、測定可能な形で示します。

**Q: 成果が不確実な場合はどうすればよいですか？**
A: 現実的な成果を示し、不確実性がある場合はその理由を説明します。
