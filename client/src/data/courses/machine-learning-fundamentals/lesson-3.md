## 📖 セクション1: クラスタリング

### クラスタリングとは

クラスタリングは、データを類似性に基づいてグループに分ける手法です。

クラスタリングの特徴：

クラスタリングの特徴として、正解データ（ラベル）が不要というラベル不要があります。また、類似したデータを同じグループに分類するという類似性、データの構造を発見するというグループの発見もあります。これらの特徴により、ラベルなしデータからも有用な情報を抽出できます。

### 医療現場での応用

医療現場での例：

クラスタリングの医療現場での例として、症状のパターンに基づいて患者を分類するという患者の分類があります。また、疾患の特徴に基づいて分類するという疾患の分類、治療反応のパターンに基づいて分類するという治療反応の分類もあります。これらの例により、クラスタリングが医療現場で活用されています。

## 📖 セクション2: 次元削減

### 次元削減とは

次元削減は、データの次元を削減する手法です。

次元削減の目的：

次元削減の目的として、高次元データを低次元で可視化するという可視化があります。また、計算量を削減するという計算効率、重要な情報を抽出するというノイズ除去もあります。これらの目的により、次元削減が有用になります。

### 主成分分析（PCA）

主成分分析（Principal Component Analysis, PCA）は、よく使われる次元削減手法です。

PCAの特徴：

主成分分析（PCA）の特徴として、データの分散を最大化する方向を見つけるという分散の最大化があります。また、高次元データを低次元に変換するという次元の削減、重要な情報を保持するという情報の保持もあります。これらの特徴により、PCAは効果的な次元削減手法となっています。

## 📖 セクション3: 異常検知

### 異常検知とは

異常検知は、正常なデータから外れた異常なデータを検出する手法です。

異常検知の特徴：

異常検知の特徴として、正常データのみで学習可能というラベル不要があります。また、異常なパターンを検出するという異常の検出、異常を見落とさないという見落としの防止もあります。これらの特徴により、異常検知が有用になります。

### 医療現場での応用

医療現場での例：

異常検知の医療現場での例として、正常範囲外の検査値を検出するという異常検査値の検出があります。また、異常な画像パターンを検出するという異常画像の検出、患者の異常な行動を検出するという異常行動の検出もあります。これらの例により、異常検知が医療現場で活用されています。
