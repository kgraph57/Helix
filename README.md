# 🏥 Medical Prompt Hub

**医療従事者のためのAIプロンプトライブラリ**

Medical Prompt Hubは、医療現場でAI（特にLLM）を活用するための高品質なプロンプトテンプレート集です。診断支援、治療計画、患者コミュニケーション、医学文献の要約など、医療業務の様々な場面で使用できるプロンプトを提供します。

## ⚠️ 重要な注意事項

このリポジトリのプロンプトは、医療従事者の業務支援を目的としたものです。AIの出力結果は必ず医療専門家が確認し、最終的な医療判断は必ず有資格者が行ってください。本プロンプトの使用によって生じたいかなる結果についても、作成者は責任を負いません。

## 📚 プロンプトカテゴリ

### 🔍 診断支援 (Diagnosis)
患者の症状や検査結果から鑑別診断を支援するプロンプト集です。

- **鑑別診断**: 症状から考えられる疾患のリストアップ
- **症状分析**: 症状の詳細な分析と重症度評価
- **検査結果解釈**: 臨床検査データの解釈支援

### 💊 治療計画 (Treatment)
エビデンスに基づいた治療計画の立案を支援するプロンプト集です。

- **治療計画立案**: 疾患に応じた治療オプションの提示
- **薬剤選択**: 患者背景を考慮した薬剤選択支援
- **投与量計算**: 体重や腎機能に基づく投与量調整

### 💬 患者コミュニケーション (Communication)
患者への説明や教育を支援するプロンプト集です。

- **患者教育**: 疾患や治療についてのわかりやすい説明
- **インフォームドコンセント**: 治療の説明と同意取得の支援
- **共感的応答**: 患者の不安や質問への適切な対応

### 📖 医学文献 (Literature)
医学論文やガイドラインの理解を支援するプロンプト集です。

- **論文要約**: 医学論文の構造化された要約
- **エビデンス統合**: 複数の研究結果の統合と評価
- **ガイドライン抽出**: 診療ガイドラインの要点抽出

### 🩺 症例分析 (Case Analysis)
臨床症例の分析と教育を支援するプロンプト集です。

- **症例提示**: 症例の構造化された記述
- **臨床推論**: 症例に基づく診断推論プロセス
- **予後予測**: 治療結果の予測と評価

## 🚀 使い方

### 基本的な使用方法

1. 目的に合ったカテゴリからプロンプトを選択
2. プロンプトテンプレートをコピー
3. 必要な情報（患者情報、症状など）を入力
4. AIモデル（GPT-4、Claude、Geminiなど）に送信
5. **出力結果を必ず専門家が確認**

### 例: 鑑別診断プロンプトの使用

```markdown
[prompts/diagnosis/differential-diagnosis.md のプロンプトを使用]

入力例:
- 患者: 45歳男性
- 主訴: 胸痛
- 症状: 労作時の胸部圧迫感、発汗、息切れ
- 既往歴: 高血圧、糖尿病
```

詳細な使用例は[examples/](./examples/)ディレクトリをご覧ください。
